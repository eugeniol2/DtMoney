<h2> Como executar 🚀  </h2>

```bash
    # Clonar o repositório
    $ git clone https://github.com/eugeniol2/DtMoney
    # Navegar para o diretório
    $ cd DtMoney
    # Instalar as package
    $ yarn
    # Rodar o projeto
    $ yarn start
```
